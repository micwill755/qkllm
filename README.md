# LLM Acceleration Project

Research and implementation of accelerated LLM training and inference techniques.

## Overview

This project explores optimization strategies for Large Language Models, from low-level kernel implementations to high-level distributed training patterns.

## Dependencies

- **qmx** - Fast matrix operations ([PyPI](https://pypi.org/project/qmx/))

## Installation

```bash
cd llm
pip install -r requirements.txt
```
